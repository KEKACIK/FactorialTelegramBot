# FactorialTelegramBot

Бот по вычислению факториала числа.

## Установка

### Go
```bash
git clone https://github.com/KEKACIK/FactorialTelegramBot.git
go mod download
go run main.go
```
### Docker Compose
```bash
git clone https://github.com/KEKACIK/FactorialTelegramBot.git
docker compose up --force-recreate --build -d
```
